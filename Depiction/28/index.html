پەیوەندیەکانی LS دەجوڵێنەوە شریتێکی سکرۆڵکردنی ئاسۆیی زیاد دەکەن لەگەڵ بەردەنگە دڵخوازەکانت لە شاشەی قوفڵ/ناوەندی ئاگانامەکانت! دەتوانیچی بکەی لەگەڵ ئەو بەردەنگانە؟ - تەلەفۆنیان بۆ بکە وە لەگەڵ ئاماژەیەکی درێژی ڕۆژنامەوانی دەتوانیت: - نامەیەکیان بۆ بنێرە بە بەکارهێنانی WhatsApp - ناردنی نامەی دەقی ئاسایی بۆیان - ئیمەیڵیان بۆ بنێرە (ئەگەر ناونیشانی ئیمەیل پێناسە کرا) .. وە هەروەها ڕۆژی لەدایک بوونیان ببینن (ئەگەر پێناسە کرا) پەیوەندیەکانی LS بژاردەی زۆری هەیە بۆ ڕێکخستن و زۆر شتی تر کە دێن، لە ئێستادا بژاردەکان ئەمانەن: هەڵبژاردنەکانی مانشێت:

 


هەمیشە لە سەرەوە - ڕێکخستنی دیمەنی پەیوەندیەکانی LS لەسەروی هەموو ڕووەکانی UI (وەک میدیا لێدەر) ئەگەر ناچالاک کرا - لە کۆتی ئاگانامەکەت دا بەخوارەوە دائەبەزێ (ئەگەر شتێکی گرنگترت هەیە بۆ بینینی سەرەوە بۆ نموونە)

بەردەنگەکانی لاپەڕە - هەستێکی جیاواز کاتێک خشاندن لە نێوان بەردەنگە زیادکراوەکانت، بەخۆڕایی تۆمارکردن نابێت وەک بژاردەی گریمانەیی پێشکەش دەکرێت، بەردەنگەکانت دابەش دەکات بەسەر لاپەڕەکاندا

پرسیار بکە پێش پەیوەندیکردن - گفتوگۆیەک پێشکەش دەکات پێش ئەوەی پەیوەندیەکەت پەیوەندی بکەیت (بۆ ئەوەی بە هەڵە پەیوەندی بە کەسەوە نەکەیت)

دۆخی تاریک - شێوازی دۆخی تاریک شێوازی نیشاندانەکەت! (لە ئێستادا تەنها بۆ ستایڵی stackview کارپێکراوە)

شەفافییەتی بینینی سەرەکی - تۆ بڕیار بدە کە بینینی تۆ چەند شەفاف بێت کە پەیوەندیەکانت لە خۆی دەپێون!

 


لە کاتی سنورداری WhatsApp لە API ی خۆیان، بۆ ئەوەی WhatsApp هەبێت بۆ ناردنی پەیامێک بە سەرکەوتوویی پێویستە پێش ئەوەی بەردەنگت زیاد بکەیت بۆ لیستی tweak (بۆ نموونە، ئەگەر ژمارەی تەلەفۆنی پەیوەندیت 054xxxxxx بێت پێویستە کۆدی وڵاتەکەت زیاد بکەیت پێش ژمارەکە) هەموو جارێک کە گۆڕانکارییەک دەکەیت لەگەڵ زانیاری بەردەنگەکەت، پێویستە دووبارە بیکەیتە لیستی درەوژەکە، پەیوەندیەکان تۆمار دەکرێن تەنها کاتێک زیاد دەکرێت بۆ لیستەکە و لە کاتی ڕاستەقینەدا نوێ ناکرێتەوە (بۆ پاراستنی بەکارهێنانی پاتری)
